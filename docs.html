Introduction

You.dl is a youtube video downloading utility that makes downloading videos a lot easier than most other platforms. You can use the platform itself to do that or create an account,
get an api key and enjoy the unlimited number of requests. You can either do a full check on all available formats for the specific video or specify a single format to check for. all of which is explained thoroughly in the Using the API section.

Creating an account and using the API is highly recommended.

We Also Have a Great CLI

Our team managed to create a CLI as well, making our service even more friendly.

Simplified CLI

The CLI itself uses the same methodology you would use in your own programming language.

  • Connect to API.
  • Get the API key.
  • Store it in Envirenment Variables.
  • Sending and receiving information to and from API.

    • Installation

    Brew
    $ brew install youdl-cli
    NPM
    $ npm install youdl-cli
    Yarn
    $ yarn install youdl-cli

    Authenticate

    $ youdl --login {username}:{password} --store

    To show the API Key

    $ youdl --key

    Get all available formats

    $ youdl --vid {video_id} --all

    Get a specific format

    $ youdl --vid {video_id} --format {format}

    The response to the CLI will be the same as with API, a nicely wrapped json object with all the video details.
